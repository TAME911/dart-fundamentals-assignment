## Assignment Description
This repository contains the solution for Programming Assignment 1
for the **Mobile Application Development** course. The assignment
focuses on learning the fundamentals of the Dart programming
language, which is used to develop mobile applications with Flutter.

## Task 1 – Number Analysis App
This program analyzes a list of integers and calculates:
- Maximum value
- Minimum value
- Sum of numbers
- Average value
The implementation uses loops, functions, and typed lists in Dart.

## Task 2 – Async Calculator App
This program implements a  Calculator class that performs
basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division
It also demonstrates:
- Object-Oriented Programming (OOP)
- Exception handling for division by zero
- Asynchronous programming using **async** and **await**
